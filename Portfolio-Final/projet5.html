Projet 5: Tutoriel en PHP  

L’unité 3 du cours ICS4U couvre les bases du langage PHP. Voici un bref tutoriel qui résume les concepts essentiels en se basant sur les connaissances en JavaScript acquises en ICS3U.

  • Les balises

    • Les balises utilisées en PHP sont <?php ... ?>

  • Les variables

    • Les variables en PHP sont très similaires à celles en JavaScript, à l'exception des quelques détails suivants. D'abord, on déclare la variable avec un signe dollar ($ma_variable = ...). Ensuite, PHP infère la valeur assignée selon un des types de données suivants: chaîne de caractères, entier, nombre décimal ou booléen.

    Pour concatener deux chaînes de caractères, ou bien une chaîne de caractères et un autre type de données, il y a deux méthodes. Soit on écrit:

    "ma variable est $ma_variable";

    soit on écrit

    'ma variable est '.$ma_variable;

  • Les tableaux

    • Les tableaux sont des séries de données que l'on crée grâce à la fonction «array()». Il y a deux types de tableaux:

    1. Les tableaux numérotés que l'on accède par indexe numérique comme
    $mon_Tableau=array(1,3,5,7);//[1,3,5,7]
    $mon_Tableau[0]; // renvoie le premier élément: 1
    $mon_Tableau[count($mon_Tableau)-1]; // renvoie le dernier élément: 7

    2. Les tableaux associatifs que l'on accède par clé comme
    $mon_Tableau1=array('un'=>1,'3'=>trois','cinq'=>5,'dernier'=>7);
    $mon_Tableau['un']; // renvoie la valeur de 'un' 1

  • Les fonctions

    • Les fonctions en PHP ressemblent fortement à celles en JavaScript, à l'exception des paramètres qui doivent être précédés d'un $ (selon la syntaxe des variables). Par exemple
    function maFonction($mon_parametre){
    ... ;
    return ... ;
    }

  • La portée

    • En PHP, la portée est plus stricte qu'en JavaScript. Une variable déclarée hors d'une fonction ne peut être accédée dans la fonction en utilisant le mot clé «global» et de même pour les variables assignées dans une fonction et accédée hors (à moins qu'elle soit retournée grâce à «return»). Aussi, il y a des variables dites superglobales. Ce sont souvent des tableaux associatifs et suivent la syntaxe $VARIABLE. Le contenu des superglobales peut être accédé n'importe où dans le fichier. Par exemple: $GLOBALS contient toutes les variables globales (pas besoin d'utiliser le mot clé «global») et $_POST contient les données d'une requête HTTP par méthode POST.

  • include()

    • La fonction include() prend l'URL vers un fichier, souvent un document HTML, et affiche sont contenu comme si tout le contenu du fichier était présent à cet endroit. Cette fonction est très utile pour des contenus tels les barres de navigation, les entêtes, les logos et tout autre contenu qui répété plusieurs fois.

En créant des serveurs, on apprend à manier le système WAMP, soit un serveur Apache, qui stocke des données avec MySQL, qui est programmé avec PHP et qui est hébergé sur Windows. UWAMP est un logiciel qui permet de gérer tous ceci sans avoir à installer et configurer chaque composant individuellement. De plus, PHPMyAdmin permet de visualiser les bases de données du serveur.
